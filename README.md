## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/amlanmohanty1/customer-trends-data-analysis-SQL-Python-PowerBI.git
cd customer-trends-data-analysis-SQL-Python-PowerBI
```

### 2. Run the Python Notebook
Open `Customer_Shopping_Behavior_Analysis.ipynb` — covers data import, exploration, cleaning, and loading into a SQL database.

### 3. Run SQL Queries
- Create a database in MySQL / PostgreSQL / MS SQL Server
- Use the notebook to load data into your database
- Open `customer_behavior_sql_queries.sql` and run the business queries

### 4. Open the Power BI Dashboard
- Open `customer_behavior_dashboard.pbix`
- Connect it to your SQL database to refresh and explore the interactive dashboard

### 5. Review the Report
Read the project report for a summary of key findings and business recommendations.
